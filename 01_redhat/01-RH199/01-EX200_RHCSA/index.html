    Red Hat Certified System Administrator (RHCSA) exam

    Puntos del exámen

    Entender el uso esencial de las herramientas

    Objetivo Estado
    Acceder a Shell prompt y ejecutar comandos con la sintaxis correcta
    Usar input-output de redirección '(>, >>,|, 2>, etc.)'
    Acceso remoto a systemas usando SSH
    Log in y switch de usuarios en obketivos multiusuario
    Archivar, comprimir, descomprimir y descomprimir archivos mediante tar, gzip y bzip2.
    Crear y editar archivos de texto
    Crear, eliminar, copiar y mover archivos y directorios.
    Crear enlaces hard y soft
    Listar, establecer y cambiar permisos estándar ugo/rwx
    Localizar, leer y utilizar documentación del sistema, incluyendo man, info y archivos en /usr/share/doc

    Crear shell scripts simples

    Objetivo Estado
    Ejecutar código condicionalmente (uso de: if, test, [], etc.)
    Uso de construcciones en bucle (for, etc.) para procesar entradas de archivos o líneas de comandos
    Procesar intputs de script ($1, $2, etc.)
    Procesar outputs de comandos de shell dentro de un script

    Operar sistemas en ejecución

    Objetivo Estado
    Boot, reinicio y apagado del sistema normalmente
    Boot con diferente objetivo manualmente
    Interrrumpir el boot para acceder al sistema
    Identificar procesos itensivos de CPU/memoria y matarlos
    Ajustar la programación de procesos
    Gestionar el ajuste de perfiles
    Localizar e interpretar los logs del sistema y diarios
    Iniciar, detener y verificar el estado de los servicios de red
    Transferir archivos entre sistemas de forma segura

    Configurar Storage local

    Objetivo Estado
    Listar, crear y eliminar particiones en discos MBR y GPT
    Crear y eliminar volúmenes físicos
    Asignar volúmenes físicos a grupos de volúmenes
    Crear y eliminar volúmenes lógicos
    Configurar sistemas para montar sistemas de archivos en el boot por ID único universal (UUID) o etiqueta
    Añadir nuevas particiones y volúmenes lógicos, e intercambiar a un sistema de forma no destructiva

    Crear y configurar sistemas de archivos

    Objetivo Estado
    Crear, montar, desmontar y utilizar sistemas de archivos vfat, ext4 y xfs
    Montar y desmontar sistemas de archivos de red mediante NFS
    Configuración de autofs
    Ampliar volúmenes lógicos existentes
    Crear y configurar directorios set-GID para colaboración
    Diagnosticar y corregir problemas de permisos de archivos

    Despliegue, configuración y mantenimiento de sistemas

    Objetivo Estado
    Programar tareas mediante at y cron
    Iniciar y detener servicios y configurar servicios para que se inicien automáticamente en el arranque
    Configurar sistemas para que arranquen automáticamente en un destino específico
    Configurar clientes de servicios horarios
    Instalar y actualizar paquetes de software desde Red Hat Network, un repositorio remoto o desde el sistema de archivos local
    Modificar el gestor de arranque del sistema

    Administración de redes (básicas)

    Objetivo Estado
    Configurar direcciones IPv4 e IPv6
    Configurar la resolución de nombres de host
    Configurar los servicios de red para que se inicien automáticamente al arrancar
    Restringir el acceso a la red mediante firewall-cmd/firewall

    Administrar usuarios y grupos

    Objetivo Estado
    Crear, eliminar y modificar cuentas de usuario locales
    Cambiar contraseñas y ajustar el tiempo de caducidad de las contraseñas de las cuentas de usuario locales
    Crear, eliminar y modificar grupos locales y pertenencias a grupos
    Configurar el acceso de superusuario

    Administración de seguridad

    Objetivo Estado
    Configurar el firewall con firewall-cmd/firewalld
    Gestión de permisos de archivos por defecto
    Configurar la autenticación basada en claves para SSH
    Establecer modos de aplicación y permisivos para SELinux
    Enumerar e identificar contextos de archivos y procesos SELinux
    Restaurar contextos de archivo predeterminados
    Gestión de etiquetas de puertos SELinux
    Utilizar configuraciones booleanas para modificar la configuración de SELinux del sistema
    Diagnosticar y resolver infracciones rutinarias de la política SELinux

    Administración de containers

    Objetivo Estado
    Buscar y recuperar imágenes de contenedores de un registro remoto
    Inspeccionar imágenes de contenedores
    Realizar la gestión de contenedores utilizando comandos como podman y skopeo
    Construir un contenedor a partir de un archivo de contenedor
    Realizar la gestión básica de contenedores como ejecutar, iniciar, detener y listar contenedores en ejecución
    Ejecutar un servicio dentro de un contenedor
    Configurar un contenedor para que se inicie automáticamente como un servicio systemd
    Adjuntar almacenamiento persistente a un contenedor
